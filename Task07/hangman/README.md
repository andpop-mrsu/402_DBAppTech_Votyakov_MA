# Игра "Виселица"

Классическая игра "Виселица" на JavaScript с использованием IndexedDB для сохранения прогресса.

## Демо

[Играть онлайн](https://mih-gif.github.io/Mih_gif.github.io/)

## Возможности

### Игровой процесс
- Угадывание букв по одной
- Возможность угадать всё слово сразу
- Визуализация прогресса через рисунок виселицы
- Ограничение на количество ошибок (6 попыток)

### Работа с базой данных
- **Автоматическое сохранение** каждой партии в IndexedDB
- **Запись всех ходов** (буквы, слова, результаты)
- **Воспроизведение сохраненных партий** с пошаговым просмотром
- **Поиск игр** по имени игрока
- **Удаление партий** из истории
- **Подробная статистика** по всем сыгранным играм
- **Экспорт статистики** в текстовый файл

###  Управление историей
1. **Список всех партий** - хронологический список всех сыгранных игр
2. **Просмотр деталей игры** - возможность посмотреть:
   - Исходное слово
   - Все сделанные ходы
   - Результат игры
3. **Воспроизведение** - пошаговое воспроизведение ходов игры:
   - Автовоспроизведение
   - Пошаговый режим
   - Пауза/возобновление
4. **Поиск** - найдите игры по имени игрока
5. **Статистика** - обзор успехов всех игроков

## Структура базы данных

База данных IndexedDB содержит три хранилища:

### 1. `words`
- Список слов для игры
- Автоматически заполняется начальным набором слов

### 2. `games`
- Информация о каждой партии:
  - ID игры
  - Дата и время
  - Имя игрока
  - Загаданное слово
  - Результат (победа/поражение)
  - Количество ошибок
  - Максимальное допустимое количество ошибок

### 3. `attempts`
- Все ходы игрока:
  - ID попытки
  - ID игры (связь)
  - Номер попытки
  - Предположение (буква или слово)
  - Тип (буква/слово)
  - Результат (попадание/промах)

##  Как играть

### Начало новой игры
1. Введите имя игрока
2. Нажмите "Начать игру"
3. Игра автоматически сохраняется в базу данных

### Просмотр истории
1. Перейдите на вкладку "Список партий"
2. Выберите игру для просмотра
3. Используйте кнопки управления воспроизведением

### Поиск игр
1. Перейдите на вкладку "Поиск игр"
2. Введите имя игрока
3. Просмотрите найденные игры

### Статистика
1. Перейдите на вкладку "Статистика"
2. Просмотрите общую статистику
3. Экспортируйте данные при необходимости

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Хранение данных**: IndexedDB (нативный API)
- **Шрифты**: Orbitron, Russo One (Google Fonts)
- **Деплой**: GitHub Pages
