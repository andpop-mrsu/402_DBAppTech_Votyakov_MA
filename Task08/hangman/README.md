# Игра "Виселица"

Классическая игра "Виселица" на JavaScript.

## Запуск

Перейдите в каталог проекта (где находится папка `public`) и запустите сервер:
   ```bash
   cd Task08\hangman
   php -S localhost:3000 -t public
   ```
   Откройте браузер и перейдите по адресу:
   http://localhost:3000/
   

## Как играть

1. Цель — угадать скрытое слово, выбирая буквы.
2. Механика — каждая ошибка добавляет элемент виселицы.
3. Конец игры — победа при полном угадывании или поражение при 6 ошибках.
4. Запуск — кнопка Новая игра.

## Режимы игры

- **Новая игра** — случайное слово, чистый лист.
- **Сохранённые игры** — архив всех партий с детальной историей ходов.
- **Информация** — правила и описание функционала.

## Сохранение и статистика

- Все партии автоматически сохраняются в SQLite базе данных.
- Доступен полный лог ходов для каждой сыгранной игры.
- В деталях игры отображаются: слово, результат, последовательность попыток.

## Функции
- Угадывание букв и слов
- Сохранение партий
- Просмотр истории игр
- Адаптивный дизайн
